# Pineapple tools
This is a IDE tools for the Pineapple language.
It include some of the features like:
- Syntax highlight
- Intellisense (on the way)

## How to package it?
```
vsce package
```

## How to install it into VSCode?
```
code --install-extension <package_name>.vsix
```
